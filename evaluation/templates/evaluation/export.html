Instructions for exporting collected evaluation data

Export via Django Admin

  1. Go to the Django Admin Panel
  2. Navigate to Evaluation section
  3. Click on Dialect Evaluations or Plausibility Evaluations
  4. Select the entries you want to export
  5. Use the export functionality to download as CSV/Excel

Export via Command Line

Run these commands in your terminal:

# Export dialect evaluations to CSV
python manage.py dumpdata evaluation.DialectEvaluation --format json > dialect_evaluations.json

# Export plausibility evaluations to CSV
python manage.py dumpdata evaluation.PlausibilityEvaluation --format json > plausibility_evaluations.json

Or use Django shell to create custom CSV exports with specific fields.
